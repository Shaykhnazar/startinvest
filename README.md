### Requirements!

_**Node version**_
```shell
    nvm install v20.15.1
    nvm use v20.15.1
    nvm alias default v20.15.1
```

## Project Setup

### Frontend

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
